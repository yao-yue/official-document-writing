## 思考
1、写总结要是自己思考转化出来的，抄总结框架没有经过思考是只从大脑里过了一遍没有留住。
2、不如选几篇好文章，直接硬背下来找找感觉。


## 案例分析
1、归纳总结  
写好总结性开头。小标题具有概括性，把分布在不同地方的相同做法归纳在一起。
要有抽象提升能力，还要有一些词句的积累。比如不充分，缺乏等等。
