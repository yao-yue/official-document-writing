## 江西遴选省直真题


## 2025-11-15真题 A卷
1、分析题
谈谈你对用好红色资源的认识和理解。500字以内，20分
2、近期，某省直机关计划召开“进一步提振消费扩大内需“工作座谈会，请你为参会领导写一份提纲做参考。600字以内，20分
3、结合省直单位工作实际，对推进公共服务均等化提对策。要求观点鲜明，对策可行，背景材料仅供参考。600字以内，20分
4、结合习近平考察江西重要讲话，结合江西发展实际，以构建具有江西特色和优势的现代化产业体系为主题写一篇策论文1000-1200字，40分


## 2025-11-15真题 B卷
1、结合省直机关工作实际，谈谈对推进作风建设常态化长效化的理解，500字以内，15分
2、为深入实施提振消费行动，拓展消费场景，你是省商务厅工作人员，请为进一步拓展消费场景提出工作要点。600字以内，15分
3、结合江西实际，就过渡期后增强脱贫地区和脱贫群众内生动力提出对策建议，供领导参阅。不超过700字，20分
4、结合江西实际，请以江西推动科技创新和产业深度融合为主题，写一篇对策性文章。1200-1500字，50分

比如：如何因地制宜发展新质生产力
